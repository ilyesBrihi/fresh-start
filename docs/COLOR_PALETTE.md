Color Palette
=============

Fresh Start uses a carefully selected color palette designed for clarity and reduced eye strain during long coding sessions.

Syntax Colors
-------------

| Element         | Color Code  | Description                                    |
|-----------------|------------|------------------------------------------------|
| Keywords        | #d9a4ffff  | Soft purple for if, for, class, def, func      |
| Names           | #fcc060ff  | Warm amber for identifiers and class names     |
| Method Names    | #7fa1ffff  | Clear blue for functions and methods           |
| Imports         | #ffcea8ff  | Peach tone for import statements               |
| Built-in Types  | #60ffa5ff  | Mint green for standard library types          |
| Comments        | #75787cff  | Muted gray for comments                        |
| Strings         | #d9ffb2e9  | Light yellow-green for string literals         |
| Docstrings      | #c3d2b5e9  | Earthy green for documentation                 |
| Numbers         | #fcbc92ff  | Salmon for numeric values                      |
| Variables       | #e9ecefff  | Clean off-white for variable names             |
| Labels          | #ffd5fcff  | Soft pink for tags and markers                 |
| Operators       | #c5ffedff  | Light cyan for operators and punctuation       |

Language Support
----------------

Currently optimized for Go, Python, TypeScript, JavaScript, JSON, and YAML.

Customization
-------------

Colors are defined in theme-colors.json and referenced throughout the theme using {{variable-name}} syntax. Modify this file to create your own color scheme.
