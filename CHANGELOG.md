Changelog
=========

All notable changes to Fresh Start will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

[2.0.0](https://github.com/ilyesBrihi/fresh-start/releases/tag/v2.0.0) - 2025-12-27
-----------------------------------------------------------------------------------

### Added

* Syntax highlighting support for Python

* Syntax highlighting support for TypeScript

* Syntax highlighting support for JavaScript

* Syntax highlighting support for JSON

* Syntax highlighting support for YAML

* Comprehensive documentation including COLOR\_PALETTE.md

* Detailed README files for project structure and theme modules

* Keywords in package.json for improved marketplace discoverability

### Changed

* Complete overhaul of syntax highlighting color scheme with new palette

* Reduced shadow effects on widgets and panels for cleaner appearance

* Updated package description to reflect multi-language support

* Enhanced LICENSE.md with precise legal terms and derivative works permissions

### Technical

* Maintained modular theme architecture with organized directory structure

* Build system continues to merge theme modules and resolve color variables

* All syntax colors now defined with distinct hex values for better code readability
